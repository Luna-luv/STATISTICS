# 01. 통계학 이해하기

```
✅ 학습 목표 :
* 통계학의 필요성에 대해 인식한다.
* 기술통계와 추론통계의 특성을 구분할 수 있다.
```
**1-1. 머신러닝 vs 전통적 통계학**
- 통계학은 [과거, 현재] 데이터를 활용해 분석 vs 머신러닝은 [ 미래 ] 예측, 모델에 기반한 분류, 선택을 목적

**1-2. 기술 통계와 추론 통계**
- 기술통계(descriptive) : 주어진 데이터의 특성을 사실에 근거하여 설명하고 묘사(중앙값, 최빈값, 평균 등 -> 산점도, 박스플롯, 히스토그램 등으로 시각화화)
- 추론통계(inferential) : 한 학급의 통계치를 통해 학교 전체 학생의 통계치 추정, 표본의 특성 -> 일반화 -> 모집단의 특성으로 '추정'

# 02. 모집단과 표본추출

```
✅ 학습 목표 :
* 모집단과 표본의 정의와 관계를 설명할 수 있다.
* 편향과 분산의 차이를 설명할 수 있다.
```
**2-1. 용어**
- 모집단(population) : 분석 대상 전체의 집합
    -> 전수조사 : 모집단의 자료 전체 조사 및 분석하여 추출
- 표본(sample) : 모집단의 일부
    -> 표본 조사 : 표본을 통해 모집단의 정보 추정, 검정
    -> 경제적, 시간적으로 유리한 이점, 모집단 전부 파악 불가능할 때도 필요

**2-2. 표본**
- 표본 오차 : 모집단과 표본의 자연 발생적인 변동(모집단 > 표본)
- 비표본 오차 : 표본 오차를 제외한 변동
- 표본추출편향 : 표본 추출 과정에 체계적인 경향이 개입되어 모집단에서 편향된 표본만 추출
- 가구 편향 : 모집단의 부분 집단 단위에서 하나의 관측시씩 추출하는 경우 크고 적은 집단이 작고 많은 집단보다 적게 추출되는 경우
- 무응답편향 : 설문에 응답하지 않는 사람이 존재(응한 사람의 대답만 표본이 됨)
- 응답편향 : 설문 형식의 문제, 응답자의 심리적 이슈에 의한 영향
    ex. 브래들리 효과 : 피조사자가 자신의 생각이나 신념을 밝히기 어려워하여 거짓된 응답을 하는 현상

**2-3. 인지적 편향**
- 확증 편향 : 믿고 있는 대로 정보를 선택적으로 받아들이고 임의로 판단
- 기준점 편향 : 분석가가 가장 처음에 접하는 정보에 지나치게 매몰
- 선택 지원 편향 : 본인이 의사결정을 내리는 순간 그 선택의 긍정적인 부분에 대해 더 많이 생각하고 그 결정에 반대되는 증거 무시
- 분모 편향 : 분수 전체가 아닌 분자에만 집중하여 현황 왜곡
- 생존자 편향 : 소수의 성공한 사례를 일반화된 것으로 인식

**2-4. 표본 편향을 최소화하기 위해**
- 단순 임의 추출방법 : like 제비뽑기, 로또당첨번호. 모집단에 대한 사전지식이 없는 경우 유용
- 계층적 표본추출방법 : 모든 구성단위에 일련번호 부여 후 일정한 간격으로 선택.
- 층화 표본 추출방법 : 모집단이 특정한 기준으로 분류가 가능할 때 쓰임. 같은 기준으로 분류된 표본에서 일정하게 뽑기
- 군집 표본 추출방법 : 모집단을 특정 기준으로 분류 후 그 중 하나의 소집단 분석.
- 복원추출법 : 처음 모집단에서 추출된 표본을 되돌려 넣고 다음 표본 추출
- 비복원추출법 : 처음 모집단에서 추출된 표본을 돌려놓지 않고 다음 표본 추출출


# 03. 변수와 척도
```
✅ 학습 목표 :
* 독립변수, 종속변수의 관계를 파악할 수 있다.
* 척도(변수의 데이터적 속성)의 종류를 설명할 수 있다.
```
**3-1. 관계계**
- 인과관계 : 독립변수와 종속변수의 기본적 관계(하나가 다른하나에 영향 미침)
- 상관관계 : 변수 간 관련성은 존재하지만 항상 인과관계를 같지는 않음
- 독립관계 : 변수 간 상관성(상관계수)가 0
- 의사관계 : 변수 간 상관성 있지만 그 상관성이 다른 변수에 의해 나타남(인과=/=상관관계인 이유)
- 양방향적 인과관계 : 두 변수가 서로 간에 인과적 영향
- 조절관계 : 독립변수와 종속변수 사이에 강하고 불확정적인 영향을 미치는 관계
- 매개관계 : 독립변수와 종속변수의 중간에서 매개변수가 개입되어 독립변수의 영향을 종속변수에 전달하는 관계

**3-2. 척도**
- 명목척도 : 조사대상의 속성이나 범주를 구분하기 위한 목적
- 서열척도 : 조사대상의 속성 크기를 측정하여 대상 간의 순서관계 측정
- 등간척도 : 서열척도가 가지고 있는 정보 + 조사대상이 가지고 있는 속성의 상대적 크기 비교 가능
- 비율척도 : 가장 많은 정보를 담을 수 있음(ex. 순서, 대상 간 상대적 크기, 절대적 기준을 통한 비율 등)


# 04. 데이터의 기술 통계적 측정

```
✅ 학습 목표 :
* 산포도의 의미를 설명하고 측정방법을 나열할 수 있다.
* 정규분포의 왜도값과 첨도값이 얼마인지 답할 수 있다.
```

**4-1. 용어**
- 평균 
    - 산술평균(기본 평균 측정 방식), 가중평균(가중치 더해서 평균 구하기), (기하적 측정에서 시작된 평균 측정 방식), 조화평균(시간적으로 변화하는 데이터에서 평균 측정)
- 중앙값 : 중앙에 위치한 값
- 최빈값 : 빈도로 측정
- 산포도 : 대푯값을 중심으로 흩어져있는 정도(범위, 분산, 편차, 사분위수 범위, 변동계수 등으로 측정)
- 왜도 : 데이터 분포의 좌우 비대칭도 표현
- 첨도 : 분포가 정규분포보다 얼마나 뾰족하거나 완만한지지



# 05. 확률과 확률변수

```
✅ 학습 목표 :
* 확률변수의 개념과 종류를 설명할 수 있다.
* 심슨의 역설을 설명하고, 발생 원인을 식별하며, 이를 해결하기 위한 방안을 도출할 수 있다.
```

**5-1. 용어**
- 비조건확률 : 아무런 조건이 없는 상황에서 사건이 일어날 확률
- 결합확률 : 표본공간 안에서 일어나는 사건 각각의 조합으로 이루어지는 확률
- 조건부확률 : 하나의 사건이 먼저 발생했다는 조건이 전제된 상황에서 또 다른 사건이 발생할 확률
- 분할 : 사건들을 모두 합했을 때 중복이 일어나지 않는 사건들의 집합
- 베이지안 이론 : 사건 발생 전 이미 가지고 있는 사전확률과 우도확률을 안다면 사후확률을 계산할 수 있다는 이론
- 확률변수 : 이산확률변수 / 연속확률변수수
